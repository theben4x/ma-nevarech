# 🍎 מה נברך? - Smart Brachot Search Engine

**מה נברך** הוא אפליקציית Web מודרנית ואינטראקטיבית לחיפוש ברכות הנהנין (ראשונות ואחרונות) הכוללת נוסח מלא ומנוקד. הפרויקט נבנה כדי להדגים שילוב של Backend מהיר עם UI מתקדם בשיטת **AI-Augmented Development**.



## 🚀 פיצ'רים מרכזיים
* **חיפוש חכם (Autocomplete):** מנגנון הצעות דינמי הכולל אימוג'ים וסינון בזמן אמת.
* **נתונים מפורטים:** מעל 100 מאכלים עם נוסח ברכה ראשונה ואחרונה מלא ומנוקד.
* **עיצוב Glassmorphism:** ממשק משתמש מודרני מבוסס שקיפויות, טשטוש (Blur) ורספונסיביות מלאה.
* **Dark Mode:** תמיכה מלאה במצב כהה/בהיר עם מעברים חלקים.
* **Backend מהיר:** מבוסס FastAPI לביצועים אופטימליים.

## 🛠 טכנולוגיות
* **Backend:** Python, FastAPI, Uvicorn.
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), Vanilla JavaScript.
* **Dev Tools:** Cursor AI (Composer & LLM integration).
* **Data Handling:** JSON-based local database.

## 🧠 מה למדתי מהפרויקט?
פיתוח האפליקציה אפשר לי להתנסות ב:
1. **Fullstack Integration:** חיבור בין לוגיקה בצד השרת (Python) לבין הצגה דינמית בצד הלקוח (JS).
2. **AI Workflow:** עבודה יעילה עם כלי AI (Cursor) לביצוע Refactoring מהיר והזרקת נתונים מורכבים.
3. **UX/UI Principles:** הטמעת עקרונות עיצוב מודרניים וחשיבה על נגישות המידע למשתמש הקצה.
