# 🍎 מה נברך? - Smart Brachot Search Engine

**מה נברך** היא אפליקציית Web מודרנית ואינטראקטיבית לחיפוש ברכות הנהנין (ראשונות ואחרונות) הכוללת נוסח מלא ומנוקד. הפרויקט נבנה כדי להדגים שילוב של Backend מהיר עם UI מתקדם בשיטת **AI-Augmented Development**.

**[Live Webapp](https://ma-nevarech.vercel.app/)**

## 🚀 פיצ'רים מרכזיים
* **חיפוש חכם (Autocomplete):** מנגנון הצעות דינמי הכולל אימוג'ים ומאכלים וסינון בזמן אמת.
* **נתונים מפורטים:** מעל 100 מאכלים עם נוסח ברכה ראשונה ואחרונה מלא ומנוקד.
* **עיצוב Glassmorphism:** ממשק משתמש מודרני מבוסס שקיפויות, טשטוש (Blur) ורספונסיביות מלאה.
* **Dark Mode:** תמיכה מלאה במצב כהה/בהיר עם מעברים חלקים.
* **Backend מהיר:** מבוסס FastAPI לביצועים אופטימליים.

## 🛠 טכנולוגיות
* **Backend:** Python, FastAPI, Uvicorn.
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), Vanilla JavaScript.
* **Dev Tools:** Cursor AI (Composer & LLM integration).
* **Data Handling:** JSON-based local database.

## 🧠 תהליך למידה ופיתוח (Learning Journey)
הפרויקט הזה הוא תוצר של תהליך למידה עצמית אינטנסיבי, שבו בחרתי לאמץ את שיטות העבודה המתקדמות ביותר בתעשייה:
* **חניכה מבוססת AI:** הפיתוח נעשה בליווי של **Gemini (LLM)** כחונך טכנולוגי, מה שאיפשר לי להבין ארכיטקטורות מורכבות תוך כדי תנועה.
* **למידה אקטיבית:** למדתי מאפס כיצד להקים Backend ב-FastAPI, לעצב ממשקי Glassmorphism ב-CSS, ולנהל גרסאות ב-Git.
* **מיומנות AI-Native:** במקום כתיבה מסורתית, התמקדתי בניהול (Orchestration) של כלי AI לפתרון בעיות, אופטימיזציה של קוד והזרקת נתונים.
